# Interactive Math Calculator & Polynomial Grapher

A Unity-based educational software designed to assist high school students in learning mathematics through interactive calculation tools and real-time polynomial function visualization.

> **Project Context:** This was developed for school project.

---

##  Key Features

* ** Polynomial Function Graphing:** Visualizes algebraic polynomial functions dynamically on a Cartesian coordinate system to help students understand how coefficients change graph behavior.
* ** Specialized Math Tools:** Includes high-school curriculum-specific calculators like GCD/LCM (EBOB/EKOK), quadratic equation root finder, and geometric distance/edge calculations.
* ** User-Friendly UI:** Clean and simplified interface custom-designed with Adobe Illustrator for intuitive navigation.
* ** Multi-Platform:** Developed in Unity using C# and targeting Android / Windows platforms.

---

##  Tech Stack & Dependencies

* **Engine:** Unity (2022.3 LTS)
* **Language:** C#
* **UI & Graphics Design:** Adobe Illustrator (Custom UI vectors)
* **Key Components:** Custom plugins/DLLs for line rendering and grid-coordinate calculations.

---

##  Project Background & Research

This project was built following a **Design and Development Research** methodology. In comparative testing with high school students:
* Group 1 (used traditional learning tools)
* Group 2 (used this Math Calculator application)

Results indicated a significant boost in student performance and problem-solving accuracy when utilizing the visual graphing tool.
